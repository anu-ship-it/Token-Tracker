# TokenPulse — Chrome Web Store Listing --- ## Extension Name TokenPulse — Token & Rate Limit Tracker --- ## Short Description (132 chars max) Live token counter and rate limit tracker for Claude and ChatGPT. Never get cut off mid-conversation again. --- ## Long Description **Never hit a token limit unexpectedly again.** TokenPulse tracks your context window and rate limits in real time — directly inside ChatGPT and Claude. No switching tabs, no guessing, no surprises. --- **What it does** A live token bar sits above the input box on both Claude and ChatGPT. As you chat, it counts tokens in real time and shows exactly how much context window you have left — before the model starts forgetting the start of your conversation. Click the toolbar icon for a full dashboard: • **Context Window** — tokens used, remaining, and percentage for the current conversation • **Claude Rate Limits** — real 5-hour session and 7-day weekly usage pulled directly from Claude's API, with exact reset countdowns • **Daily Usage History** — automatic tracking of your peak token usage per day, across the last 60 days. No setup needed — it records silently as you chat • **Smart Notifications** — get alerted at 75%, 90%, and 100% before you hit a wall. Configurable thresholds, zero noise --- **Works on both platforms** Most token trackers only work on Claude. TokenPulse works on both Claude and ChatGPT — one extension, two platforms, consistent experience. --- **Built for power users** If you write long documents, do deep research, run coding sessions, or just have long conversations — you've hit the context wall before. TokenPulse gives you the visibility to work right up to the limit without ever being caught off guard. --- **Privacy first** All data is stored locally on your device. TokenPulse makes no external requests except to Claude's own API (using your existing session) to fetch your rate limit data. No accounts, no tracking, no analytics. Open source — full code transparency. --- **Features** ✓ Live in-page token bar on Claude and ChatGPT ✓ Real Claude API rate limits — 5-hour session and 7-day weekly ✓ Exact reset countdowns ✓ Automatic daily usage history (no setup) ✓ Smart threshold notifications at 75%, 90%, 100% ✓ Consistent teal dark theme ✓ Zero configuration required --- Built by Anoop Kumar and Mansi Rathore · Alpha --- ## Category Productivity ## Language English --- ## Privacy Policy (host this at a public URL) TokenPulse Privacy Policy TokenPulse is a browser extension that tracks token and rate limit usage for Claude and ChatGPT. Data collection: TokenPulse does not collect, transmit, or store any personal data on external servers. All usage data (token counts, daily history, settings) is stored locally in your browser using Chrome's built-in storage API and never leaves your device. API access: On Claude, TokenPulse fetches rate limit data from Claude's own API (claude.ai/api/organizations) using your existing browser session. No credentials are stored or transmitted to any third party. No analytics: TokenPulse contains no analytics, tracking pixels, or third-party scripts of any kind.
